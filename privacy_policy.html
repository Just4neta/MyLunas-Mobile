📷 Face Data Collection, Use and Retention

What face data does the app collect?
MyLUNAS Mobile itself does not directly collect, store, or process any face data on the user's device. The Application provides a WebView interface to MyDEX, an internal web-based system hosted entirely on Lumut Naval Shipyard Sdn. Bhd.'s private internal servers. Any face capture or recognition occurs within the MyDEX web system — not within the Application itself.

Planned uses of face data:
Face data accessed through the MyDEX internal web system is used exclusively for:

  • Staff attendance verification
  • Secure access control within LUNAS premises

No other use of face data is planned or conducted.

Will face data be shared with third parties?
No. Face data is processed and stored solely on Lumut Naval Shipyard Sdn. Bhd.'s private internal servers located within Malaysia. Face data is never shared with any third party, vendor, or external service provider.

Where is face data stored?
Face data is stored exclusively on Lumut Naval Shipyard's secure internal server infrastructure. The Application stores zero face data on the user's device at any time.

How long will face data be retained?
Face data is retained in accordance with Lumut Naval Shipyard Sdn. Bhd.'s internal Human Resources and Security Data Retention Policy. Retention periods are reviewed periodically and data is deleted when no longer required for operational purposes.

Summary:
MyLUNAS Mobile functions solely as a browser interface (WebView wrapper) for internal LUNAS web systems. All face data processing is handled server-side by the organisation's own infrastructure. The Application has no access to, and does not store, any biometric or face data on the device.
